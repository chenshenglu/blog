hexo基本操作

发表于 2015-03-04

HEXO基本操作

更换主题

  1. 找到喜欢的主题
  2. 复制git地址
  3. 执行代码
  4. 更改yml配置的主题
    git clone gitaddr theme/themename

发布到GITHUB

  1. 下载安装gitbash
  2. 生成sshkey并绑定到github账户
  3. hexo g & hexo d
  4. 上传成功后在setting页面获得page的地址，直接访问发现css的格式不对，url要设置为发布的地址，root设置为文件目录

    修改 _config.yml 中的 url 和 root 設定：
    url: http://chenshenglu.github.io/blog
    root: /blog/

  5. 访问正常

MarkDown基本语法

发表于 2015-03-04

MarkDown语法简洁，下面只整理一些常用的：

1.标题

语法如下：

#一级标题
#二级标题
######六级标题

效果如下：

一级标题

二级标题

六级标题

2.列表

语法如下：

* 选项1
* 选项2
* 选项3

1. 选项1
2. 选项2
3. 选项3

效果如下：

无序列表

  • 选项1
  • 选项2
  • 选项3

    有序列表

  1. 选项1
  2. 选项2
  3. 选项3

3.引用

只需要在文本前加入>这种尖括号（大于号）即可
语法如下：

> 这是一个引用

效果如下：

这是一个引用

4.图片和链接

语法如下：

图像:![百度logo](https://www.baidu.com/img/bdlogo.png)
链接:[HEXO](http://hexo.io/)

5.粗体和斜体

Markdown 的粗体和斜体也非常简单，用两个包含一段文本就是粗体的语法，用一个包含一段文本就是斜体的语法。
语法如下：

**这是粗体**
*这是斜体*

效果如下：
这是粗体
这是斜体

6.表格

表格直接采用html的语法









NAME ID MEMO
aaron
082512
“it is my name”

7.代码框

语法如下：

 printf(“hello world”);

printf("hello world");

8.分割线

3个”*”号

9.其它

反斜杠用于输出MD本身的代码
